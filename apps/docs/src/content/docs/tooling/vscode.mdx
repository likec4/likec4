LikeC4 has official [extension for VSCode](https://marketplace.visualstudio.com/items?itemName=likec4.likec4-vscode) - open-source and available on [github](https://github.com/likec4/likec4/tree/develop/packages/vscode).  
The extension provides:

- Validation and error reporting
- Semantic syntax highlighting
- Live Previews (and editing)
- Code completion and navigation 
- Resolve references (like `find all references`, `go to definition` .. )
- "Safe" renames
- Hover information

Extension is universal and can run in the browser.
